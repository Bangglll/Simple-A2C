# Simple-A2C
<br/>
We created A2C for bounding box detection.<br/>
<br/>
1.simple box detection <br/>
state : x1, y1, x2, y2 [4]<br/>
action : move x1 (move left and right) [2]<br/>
reward : IoU<br/>
<br/>
2. simple box detection float<br/>
state : x1, y1, x2, y2 [4]<br/>
action : move x1, y1, x2, y2 [4]<br/>
reward : IoU<br/>
